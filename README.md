# 说明

## 功能

- **按键记录**：捕获键盘输入，包括字母、数字、符号、功能键等。
- **日志轮转**：每小时自动创建新的日志文件（文件名格式 `keylog_YYYYMMDD_HHMMSS.txt`）。
- **隐藏运行**：记录程序可无窗口运行（通过启动器）。
- **日志查看器**：
  - 自动扫描当前目录下的所有记录文件。
  - 按时间顺序列出文件。
  - 查看文件内容。
  - 支持关键词搜索。

---



## 使用方法

### 1. 启动键盘记录
- 双击 `launcher.exe`，它将在后台启动 `sender.exe`（无窗口）。
- 启动器会显示进程ID，可通过任务管理器结束进程来停止记录。
- 记录数据将保存在当前目录下按小时生成的 `keylog_*.txt` 文件中。

### 2. 查看日志
- 双击 `viewer.exe`，程序会自动扫描当前目录下的所有记录文件。
- 按提示输入文件编号查看内容，或输入 `s` 搜索关键词，输入 `q` 退出。

### 3. 手动停止记录
- 在任务管理器中找到 `sender.exe` 进程并结束，或按 `Ctrl+C`（如果在前台运行）。
